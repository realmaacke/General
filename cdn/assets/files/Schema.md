## Att göra varje vecka

Planera i 2 veckors per gång

```
Vecka(45) {
    Vteam{
        Föreläsning Tisdag 15:00
        Föreläsning Torsdag 15:00

        Starta på SDS
    }
    Matmod {
        Föreläsning Onsdag 10:15

        Starta på Assignment 1
    }
    DevOps {
        Föreläsning Måndag 13:00 - Klar

        Läs boken 1 del utav 6 (Del 1)
        Börja på KMOM01
    }
    Itsec {
        Föreläsning Måndag 10:00 - Klar

        Kmom01 Quiz
        Kmom01 referat göra klart.
    }
}

Vecka(46) {
    Vteam{
        Föreläsning Torsdag 15:00
        Föreläsning Måndag 15:00

        Fortsätt med SDS
    }
    Matmod {
        Föreläsning Onsdag 10:15

        Fortsätt med Assignment 1§
    }
    DevOps {
        Bokcirkel Onsdag 13:00

        Läs boken 1 del utav 6 (Del 2)
        Göra klart kmom01
    }
    Itsec {
        Föreläsning Måndag 10:00

        Förfallodatum kmom01 quiz och kmom01 så det måste vara klart tills på onsdag!
    }
}
```
